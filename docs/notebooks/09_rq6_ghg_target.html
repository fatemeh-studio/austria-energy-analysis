RQ6 · Is Austria’s GHG Trajectory On Track for Its 2030 Emissions Target?

Question: Is Austria on track to meet its binding 2030 greenhouse-gas target — the EU Effort Sharing Regulation’s −48% vs 2005 for non-ETS sectors (Reg 2018/842, raised from −36% by Reg 2023/857)?

Strictly the economy-wide non-ETS emissions question. The renewable-electricity trajectory is RQ5, kept separate by design — one is an electricity-mix question, this is an emissions question.

Data

Why a dedicated series. The −48% target binds only the ~63% non-ETS slice (transport, buildings, agriculture, small industry, waste), and that slice cannot be derived from Eurostat’s CRF sectors — the ETS/non-ETS boundary cuts across them. So the verdict uses the EEA’s Effort Sharing series (esr_emissions), not the TOTX4_MEMO total (that’s the −28% EDA context, a different metric).

esr_emissions — EEA non-ETS emissions, 2005–2024, Mt CO₂-eq — carries the trajectory; the binding AEA path 2021–2030 (data/reference/austria_esr_aea.csv, from Commission Implementing Decisions 2020/2126 → 2023/1319 → 2026/895) carries the target. The series is heterogeneous by construction — ESD/AR4 accounting through 2020, ESR/AR5 from 2021 — so every reduction anchors to the official AR5 2005 baseline (56.99 Mt), keeping emissions and target on one basis.

Method — log-linear (constant-%) OLS + four-window fan

Emissions decline at a roughly constant rate, so the trend is fit by OLS on log emissions — the bounded-below analogue of RQ5’s logit: emissions can’t go negative, and a constant-% path never crosses zero. A constant-absolute (linear) fit is kept alongside as a robustness check. The fit is extrapolated to 2030 with a 95% prediction band and read against the −48% target (29.6 Mt) and the binding AEA path. With only n = 6 points in the primary window, standard errors are plain — HAC would be false precision at this sample size, the same call as RQ5.

Four forks settled from the data (not assumed):

Fork Options Decided
Functional form constant-absolute (linear) vs constant-percent (log-linear) Log-linear — emissions bounded below at 0; linear kept as robustness
Trend window how far back the fit starts; 2020–23 carries COVID + energy-crisis distortion Fan of four (2005–24, 2013–24, 2019–24, 2021–24); primary 2019–24
Target representation bare −48% endpoint vs the full binding AEA path AEA path — richer, and resolves the basis question (AR5, absolute Mt)
Accounting basis the file’s AR4/ESD 2005 vs the official AR5/ESR 2005 baseline AR5 (56.99 Mt) — matches the target’s basis

The series is flat 2005–2019 then drops only after 2020, so the verdict is window-dependent and partly shock-driven — the four-window fan makes that explicit rather than hiding it behind one slope.

2 · Trend fits — log-linear (primary) + linear (robustness)

Code 
# log-linear (primary) + linear (robustness) trend fits
# across four windows. Log-linear => constant %/yr decline; linear => constant Mt/yr.

WINDOWS = [(2005, 2024), (2013, 2024), (2019, 2024), (2021, 2024)]
T0 = 2005  # shared time origin for all fits
T2030 = 2030 - T0

fits = {}  # window -> fitted models (kept for section 3)
rows = []
for y0, y1 in WINDOWS:
    w = annual[(annual.year >= y0) & (annual.year <= y1) & annual.mt_co2e.notna()]
    t = (w.year - T0).to_numpy()
    X = sm.add_constant(t)
    e = w.mt_co2e.to_numpy()

    loglin = sm.OLS(np.log(e), X).fit()  # log-linear: emissions bounded below at 0
    lin = sm.OLS(e, X).fit()  # linear: robustness comparison
    fits[f"{y0}–{y1}"] = {"loglin": loglin, "lin": lin, "w": w}

    loglin_2030 = float(np.exp(loglin.params[0] + loglin.params[1] * T2030))
    lin_2030 = float(lin.params[0] + lin.params[1] * T2030)
    rows.append({
        "window": f"{y0}–{y1}",
        "n": len(w),
        "decay_%/yr": 100 * (np.exp(loglin.params[1]) - 1),
        "loglin 2030 (Mt)": loglin_2030,
        "linear 2030 (Mt)": lin_2030,
        "% vs 2005": 100 * (loglin_2030 / BASELINE_2005_MT - 1),
        "gap vs target (Mt)": loglin_2030 - TARGET_2030_MT,
    })

results = pd.DataFrame(rows).set_index("window").round(2)
print(f"Target 2030: {TARGET_2030_MT:.2f} Mt   (−48% vs {BASELINE_2005_MT:.2f} Mt, AR5 basis)\n")
display(results)

3 · Extrapolate the primary window (prediction band)

Code 
# extrapolate the primary window forward, plus the full-window fan.
# log-linear fit, so work in log space and exponentiate back to Mt.
PRIMARY = "2019–2024"
years_fwd = np.arange(2019, 2031)  # fit start → 2030
t_fwd = years_fwd - T0
Xf = sm.add_constant(t_fwd)

m = fits[PRIMARY]["loglin"]
pred = m.get_prediction(Xf).summary_frame(alpha=0.05)  # 95% intervals, in log space
proj = pd.DataFrame({
    "year": years_fwd,
    "mean": np.exp(pred["mean"]),  # back to Mt
    "lower": np.exp(pred["obs_ci_lower"]),  # prediction interval (new obs)
    "upper": np.exp(pred["obs_ci_upper"]),
})

p2030 = proj[proj.year == 2030].iloc[0]
print(f"Primary window {PRIMARY}: 2030 central estimate {p2030['mean']:.1f} Mt "
      f"(95% prediction interval {p2030['lower']:.1f}–{p2030['upper']:.1f} Mt)")
print(f"Target {TARGET_2030_MT:.1f} Mt — clears target within band? "
      f"{'yes' if p2030['lower'] <= TARGET_2030_MT else 'no'}")

# all four windows projected to 2030 (central lines, for the fan)
fan = {}
for label, f in fits.items():
    fan[label] = np.exp(f["loglin"].params[0] + f["loglin"].params[1] * (years_fwd - T0))

display(proj.round(2))

4 · Gap scorecard — required vs recent pace

Code 
latest_year = int(esr.year.max())  # 2024
latest_mt = float(esr.loc[esr.year == latest_year, "mt_co2e"].iloc[0])
n_left = 2030 - latest_year  # 6 years

# required constant-% cut from 2024 to land exactly on target in 2030
req_decay = (TARGET_2030_MT / latest_mt) ** (1 / n_left) - 1  # fractional/yr

# recent achieved pace = the 2019–24 fitted decay (primary window)
recent_decay = np.exp(fits["2019–2024"]["loglin"].params[1]) - 1

print(f"From {latest_year} ({latest_mt:.1f} Mt) to 2030 target ({TARGET_2030_MT:.1f} Mt):")

Finding

Finding (RQ6). On its post-2019 trend, Austria’s non-ETS emissions are projected to reach ≈36.8 Mt by 2030 (95% prediction interval 29.8–45.4 Mt) — roughly 7 Mt above the binding −48% Effort Sharing target of 29.6 Mt, and short of it across all four trend windows tested (gap +2.6 to +13.9 Mt). Closing it would require cutting emissions at about −6%/yr, ~2.2× the recent pace, and the only window that nearly reaches the target leans entirely on energy-crisis-driven reductions whose permanence is unproven.
